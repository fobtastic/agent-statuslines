# agent-statuslines

Statusline configs and scripts for AI CLI agents.

| Agent | Folder | Description |
|-------|--------|-------------|
| [Antigravity CLI](antigravity/) | `antigravity/` | Custom JS statusline renderer with git, PR, and context telemetry |
| [Claude Code](claude-code/) | `claude-code/` | ccstatusline layout config (TokyoNight powerline theme) |

See each subfolder's README for setup instructions.
